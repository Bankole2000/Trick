# TRICK ENGINE GAME SPECIFICATION

# SPELLS

## Version 1.0

### Classification

Epistemic Game Type:
Language Action Game

Primary Psychological Focus:
Influence

Core Question:

> How do words change reality?

---

# Design Philosophy

SPELLS is based on the idea that language does not merely describe the world.

Language can alter:

* beliefs
* decisions
* relationships
* institutions
* future actions

In this game, statements are treated as spells.

A spell succeeds when it produces effects.

---

# Psychological Model

SPELLS trains:

* persuasion awareness
* rhetoric analysis
* influence detection
* strategic communication

The central failure mode is:

> Believing that words are powerless.

---

# Deck Characteristics

Cards contain speech acts rather than facts.

Examples:

* Promise
* Warning
* Threat
* Request
* Prediction

Example statements:

* "I will repay you tomorrow."
* "This will be dangerous."
* "Trust me."

---

# Core Mechanic

Players gain points not for truth but for measurable impact.

After a statement is played:

Listeners choose responses.

The speaker scores based on behavioral change.

Examples:

* changing votes
* changing alliances
* changing plans
* changing beliefs

---

# Scoring

Behavior Change:
+3

Belief Change:
+2

Alliance Change:
+2

Failed Influence:
0

Influence Backfire:
-2

---

# Win Condition

Highest Influence Score.

---

# What It Models

SPELLS emphasizes:

U → L

Language as a causal force.

---

# Ideal Outcome

Players become aware that:

* language shapes behavior
* narratives compete
* persuasion and truth are separate variables

The game teaches responsible use of influence.
